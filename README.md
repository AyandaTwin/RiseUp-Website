## Project Overview
This project involves the development of a professional website for RiseUp Youth Foundation, a non-profit organisation based in Johannesburg. The website aims to provide information about the organisation, its programmes, and allow users to engage through enquiry and contact forms.

---

## Website Goals and Objectives
- Increase awareness of youth development programmes  
- Encourage volunteer participation  
- Attract donations and sponsorships  
- Provide accessible and structured information  

---

## Key Features and Functionality
- Five functional web pages (Home, About, Programmes, Enquiry, Contact)  
- Navigation system linking all pages  
- Enquiry and Contact forms with validation  
- Structured and user-friendly layout
